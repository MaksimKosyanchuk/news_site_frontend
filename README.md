# Daily Buzz

Simple new site project

## Technologies
- HTMl
- CSS, SCSS
- ReactJS
- JsonWebToken
- Dotenv

## Functionality

- All news screen
- Profile screen
- New screen
- Login/Register screeen
- Saved posts

## Install and run

```bash
git clone https://github.com/MaksimKosyanchuk/news_site_frontend
cd news-site
```
set server link into `REACT_APP_API_URL` variable in .env file

```bash
npm install
npm start
```
